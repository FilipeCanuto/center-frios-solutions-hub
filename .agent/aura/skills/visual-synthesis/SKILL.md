# Protocolo Aura: Engenharia Visual

Ao criar ou recriar uma logomarca:

1. **Visual Metaphors**: Gere 3 conceitos metafóricos baseados na estratégia (ex: "A união entre fluidez e solidez").
2. **Iterative Generation**: Use modelos de imagem para exploração visual. Converta os melhores conceitos em código SVG puro para garantir escalabilidade infinita.
3. **Geometric Grid**: Aplique princípios de proporção áurea e grids geométricos no código SVG para garantir o equilíbrio visual das marcas "Big Tech".
4. **Versatilidade**: Entregue versões em:
   - Full Color (Primária)
   - Monochrome (Preto/Branco)
   - App Icon (Simplificada para tamanhos pequenos)

**Regra de Ouro**: Se o logo não puder ser desenhado à mão por uma criança em 5 segundos, simplifique-o (Princípio da Nike/Apple).
